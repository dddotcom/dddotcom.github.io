Projects

This is a selection of some projects I've been working on recently. Take a look at my github to see more of what I've been up to.

Personal Website

Contributors: (solo) @dddotcom

What is it? A responsive, personal static site featuring descriptions of and photography from my adventures in the PNW.

Technologies: HTML, CSS, JavaScript, Image HTML and CSS minifiers, JS uglifier

My role: Responsive design and styling, images, deployed on Github pages, implemented of minify and uglify script to generate a "dist" folder

Yoshi Weather

Contributors: (team) @dddotcom, @jtferns

What is it? A white-pomeranian-named-Yoshi themed static website that shows you weather data for a set number of locations. Weather data can also be viewed for a specific zipcode. Inspired by Doge Weather.

Technologies: HTML, CSS, JavaScript, OpenWeatherMap API,Surge.sh

My role: Design and styling, images and artwork, implemented API calls and zip code based weather lookup

Quite Contrary Blooms

Contributors: (solo) @dddotcom

What is it? A responsive, static site for a small flower business that uses social media and word of mouth for growing its customer base and securing orders. The site features the florist's portfolio and a contact page.

Technologies: HTML, CSS, Bootstrap , Heroku, Node.js, SendGrid

My role: Customer consultation, responsive design and styling, implemented Node server and integration with SendGrid, deployed on Heroku
